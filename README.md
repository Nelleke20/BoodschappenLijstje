# Boodschappenlijstje

A collaborative shopping list app for households вЂ” build together, shop together.

## Features

- **Real-time collaboration** вЂ” all household members see changes instantly via Firestore
- **Household management** вЂ” create a household or join one with a 6-char invite code
- **Smart categorization** вЂ” items are auto-categorized based on Dutch grocery keywords
- **Templates** вЂ” 5 built-in templates (Weekboodschappen, BBQ, Ontbijt, etc.) + save your own
- **Quick add** вЂ” type an item name and press enter
- **Swipe to edit/delete** вЂ” swipe left on any item
- **Sort & group** вЂ” by category (grouped), date added, alphabetical
- **Offline-aware** вЂ” works with Firestore offline cache
- **Dutch language** throughout

## Setup

### 1. Create a Firebase Project

1. Go to [console.firebase.google.com](https://console.firebase.google.com)
2. Create a new project
3. Enable **Authentication** в†’ Email/Password
4. Create a **Firestore Database** (start in production mode)
5. Deploy security rules (see below)

### 2. Configure FlutterFire

```bash
# Install FlutterFire CLI
dart pub global activate flutterfire_cli

# Configure (run from project root)
flutterfire configure
```

This generates `lib/firebase_options.dart` with your project's credentials. Replace the placeholder file.

### 3. Deploy Firestore Security Rules

```bash
# Install Firebase CLI
npm install -g firebase-tools
firebase login

# Deploy rules
firebase deploy --only firestore:rules
```

Or copy `firestore.rules` to the Firebase Console manually.

### 4. Add Firebase Config Files

**Android:** Place `google-services.json` in `android/app/`

**iOS:** Place `GoogleService-Info.plist` in `ios/Runner/`

These are generated by the Firebase Console when you add apps to your project.

### 5. Run the App

```bash
flutter pub get
flutter run
```

## Data Models

See `lib/models/` for full Dart definitions. Key relationships:
- `User` в†’ belongs to one `Household`
- `Household` в†’ has many `Users`, `ShoppingList`s
- `ShoppingList` в†’ has many `ShoppingItem`s
- `ShoppingTemplate` в†’ belongs to `Household` (or is default)
